Project Information

  • Project Name: Taiwan Stock Intelligence Platform
  • Project Type: Team Project
  • Topic: Web Services, Database, and Stock Data Analysis
  • Purpose: Build a web-based platform for Taiwan stock data analysis and visualization.

System Architecture

The system uses Flask as the web server. After receiving a stock code or stock name, it searches the TWSE stock list, downloads historical trading data, stores the data into SQLite, and uses Pandas to generate technical indicators and rule-based analysis.

  • User Browser
  • Flask Web Server
  • TWSE Stock List API
  • Historical Stock Data Crawler
  • SQLite Database
  • Pandas Analysis Module

Main Functions

  1. Load stock codes and stock names directly from TWSE.
  2. Search stocks by code or name.
  3. Download historical stock trading data.
  4. Store stock data in SQLite database.
  5. Calculate MA5, MA20, daily return, volume ratio, and volatility.
  6. Generate trend score and analysis reasons.
